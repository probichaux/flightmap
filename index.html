How to use FlightMap

Enter flights in the sidebar, one per line. Each line is an origin-destination pair using ICAO codes (KJFK), IATA codes (JFK), or FAA identifiers (FFC).

Separate the pair with a dash, comma, space, or arrow:

JFK-LHR
LAX NRT
KJFK, EGLL
SFO -> SIN

Upload a CSV or text file with two columns (origin and destination). The first row is treated as a header and skipped. Comma, tab, semicolon, and space delimiters are auto-detected.

Look up by tail number fetches the last 120 days of flights from OpenSky Network for a US aircraft registration (e.g., N12345). You need a free OpenSky account — add your API credentials under OpenSky Settings at the bottom of the sidebar.

Plot Flights (or Cmd/Ctrl+Enter) validates the codes and draws great-circle routes on the map.

Distance Units switches between nautical miles (default) and kilometers. Changing units instantly updates all displayed distances.

Export PNG downloads the current map view as an image.
